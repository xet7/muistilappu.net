1. TIETOKONE => KÄYTTÖJÄRJESTELMÄ => SOVELLUKSET

  • Tietokone: Läppäri, pöytäkone, älykännykkä, palvelintietokone, äly-TV
  • Läppäri käyttöjärjestelmä
    1. Windows (Microsoft seuraa)
    2. Mac: macOS (Apple seuraa)
    3. Chromebook: ChromeOS (Google seuraa)
    4. Linux (ei seurantaa). Jos tietokoneessa ei toimi uusin Windows/macOS/ChromeOS, niin asentamalla Linux saa uusimmat käyttöjärjestelmä ja sovellus päivitykset joissa uusimmat tietoturva korjaukset
  • Tieto ja tiedosto eli Bitti: 0 tai 1, MB megabitti, GB gigatavu, TB teratavu eli 1000 GB, minkä kokoinen tiedosto voi olla. Tiedosto voi olla: Teksti, kuva, video, käyttöjärjestelmä, sovellus, jne. Videot vie eniten tilaa.
  • Tiedontallennusväline johon tieto (bitit 0 tai 1) tallennetaan
    1. Magneettinen tai ei-magneettinen kohta: Kovalevy (HDD helppo särkymään jos pudottaa, tai SSD kestävämpi), C-kasetti ääni tai C64-tiedostot, disketti
    2. Onko kuoppa laserilla tms poltettu, vai ei kuoppaa: DVD, Blue-Ray, CD
  • Tiedon siirto väline:
    1. Valojohto: Merikaapelit ja valokuitu maan alla (bitti: on valo, tai ei valoa)
    2. Sähköjohto (bitti: on sähko, tai ei sähköä)
  • Tietoliikenneverkko
    1. Internet: Maailmanlaajuinen verkko: Merikaapelit valokuitu => Valokuitu maan alla => Kaapelimodeemi tai Matkapuhelinverkko
      1. Verkkotunnus
      2. Verkkosivu
      3. HTML
      4. Verkkoselain
        1. Chromium: Google Chrome (Google seuraa), Microsoft Chromium Edge (Microsoft seuraa), Brave (paras mainospoisto), Opera. Nämä Windows/Linux/Mac
        2. Mozilla Firefox (Mozilla seuraa): Iceweasel (Debian, ei seurantaa), WaterFox (ei seurantaa)
        3. Safari, macOS ja iOS
        4. Microsoft: Internet Explorer => Edge => Microsoft Chromium Edge uusin
      5. Hakukoneet
        1. Google.fi
        2. DuckDuckGo.com
        3. fi.wikipedia.org
      6. Merikaapelit
      7. Palvelinkeskus / Datakeskus. Internet on jonkun muun tietokone ja tallennustila.
    2. Matkapuhelinverkko
      1. Teleoperaattori kuten Elisa, DNA, Telia
      2. Matkapuhelin
      3. Android (Puhelimen valmistaja, Google ja sovellukset seuraa)
      4. iOS (Apple ja sovellukset seuraa), iPhone/iPad "Etsi" sovellus jolla voi etsiä esim. avaimet ja lompakon jossa Applen AirTag etsintälaite avaimenperällä tai ilman
      5. Ubuntu Touch (ei seurantaa)
